Lista zadań, cyfry są przyporządkowane numerom keszy.

Python

1. Napisz program, który zapyta o twoje imię a następnie je wydrukuje, np. Cześć Janek!

2. Napisz program, który będzie liczył, ile znaków jest w danych mu wyrazach/słowach.

3. Napisz program, który będzie stwierdzał, czy podana przez użytkownika liczba jest większa od 100.

4. Napisz program, który będzie reagował na jedno wybrane powitanie, np. UŻYTKOWNIK cześć PROGRAM Hej
użytkownik może wypowiedzieć się tylko raz.

5. Napisz program (kalkulator), który będzie obsługiwał dodawanie, odejmowanie, mnożenie, dzielenie liczb podanych przez użytkownika. Na samym początku, program ma zapytać użytkownika o typ działania (dodawanie, odejmowanie, mnożenie, dzielenie).

HTML

6. Napisz stronę html która będzie zawierać favicon, nagłówek, przynajmniej jedno zdjęcie i tabelę.

7. Napisz menu poziome w html przy użyciu pliku ccs

8. Napisz w html podstronę do logowania na stronie www.

Prac z html nie publikujemy w internecie, kod "wyślij rozwiązanie".

Jeżeli chcemy wysłać kilka plików, dodajmy ich kody do pola rozwiązania, oddzielając nazwą pliku
np.##index.html##kod..
