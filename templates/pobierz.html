Python

Pythona instalujemy z oficjalnej strony poprzez poniższe linki, lub poprzez wejście na oficjalną stronę. Aplikacja jest bezpieczna i można ją swobodnie pobrać.

Windows 32-bitowy

MacOS 64-bitowy

Kod w pythonie możemy pisać w nawet w notatniku komputerowym, lecz ważnym jest żeby zapisać go w rozszerzeniu py. Żeby zobaczyć czy nasz program działa należy załączyć go najlepiej przez command line ze ścieżką dostępu do naszego pliku, ponieważ poprzez launcher wyłącza się po dokonaniu operacji.

Jeżeli chcemy odpalać poporzez command line dodatkowo przy instalacji trzeb kliknąć "add to path" co przedstawiaponiższy obrazek.

HTML

Tutaj nie potrzebujemy specjalnego oprogramowania lecz jedynie komputera, z notatnikiem. Zapisujemy w formacie .html. Jeżeli chodzi o css to również nie potrzebujemy dodatkowego oprogramowania, lecz notatnika i zapisu plik w formacie .css.
